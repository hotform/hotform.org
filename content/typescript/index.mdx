# TypeScript

HotForm is written in TypeScript. You can add TypeScript definitions for HotForm by using the CLI:

### npm

```shell
npm i -D @hotform/types
```

### yarn

```shell
yarn add -D @hotform/types
```

To see all of the types available and their generics look at our <a rel="noreferrer" href="https://github.com/hotform/hotform/tree/master/packages/hotform-types" target="_blank">TypeScript definition package</a>.
