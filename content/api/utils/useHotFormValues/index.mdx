### Type Definition

`useHotFormValues: <T>(config: Pick<HotFormConfig<T>, 'hotField' | 'initialSchema' | 'onInvalid' | 'onValid'>) => UseHotFormValuesReturnType<T>`

### Return Value

`useHotFormValues` returns an array with exactly two values:

- `HotFormValues<T>` (object)
  - <GatsbyLink to="/api/HotFormConfig#initialSchema">`currentSchema`</GatsbyLink> (object)
  - `runValidityEvents` (boolean)
  - `submitting` (boolean)

- `React.Dispatch<HotFormAction<T>>` (function)
